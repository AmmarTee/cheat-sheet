# Contributing

This is a personal reference repo. Open a PR if you find a technical error or a useful one-liner. Ensure all commands are tested for CLI accuracy.

## Guidelines

- Test commands before submitting
- Include version-specific notes where relevant
- Use realistic example values (VMIDs, hostnames)
- Add inline comments for complex flags
- Keep formatting consistent with existing guides

## PR Process

1. Fork the repo
2. Make your changes
3. Test on actual systems
4. Submit PR with clear description

## Issues

Report technical errors or suggest improvements via GitHub Issues.
